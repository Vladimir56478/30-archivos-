# Guía de Uso

## Estructura del Proyecto

Los ejercicios están organizados en 30 archivos Python, cada uno contiene un ejercicio específico con su solución.

## Cómo usar este repositorio

### 1. Clonar el repositorio
```bash
git clone https://github.com/tu-usuario/30-archivos.git
cd 30-archivos
```

### 2. Ejecutar un ejercicio
```bash
python ejercicio_01.py
```

### 3. Ver la solución
Cada archivo contiene comentarios explicando el ejercicio y la solución implementada.

## Ejercicios por Categoría

### Nivel Básico (Ejercicios 1-5)
- Condicionales simples
- Operadores lógicos
- Manejo de variables

### Nivel Intermedio (Ejercicios 6-20)
- Bucles (for y while)
- Contadores y acumuladores
- Lógica con múltiples condiciones

### Nivel Avanzado (Ejercicios 21-30)
- Listas y arrays
- Búsqueda y filtrado de dados
- Algoritmos (números primos, divisores)

## Tips para aprender

1. Primero intenta resolver los ejercicios sin ver el código
2. Luego compara tu solución con la proporcionada
3. Experimenta modificando los ejercicios
4. Intenta optimizar el código

## Preguntas Frecuentes

**P: ¿Cómo puedo modificar un ejercicio?**
R: Puedes abrir cualquier archivo con tu editor de texto favorito y hacer los cambios que desees.

**P: ¿Hay una versión más fácil de estos ejercicios?**
R: Los primeros 5 ejercicios son los más básicos. Comienza por ahí.

**P: ¿Cómo puedo contribuir?**
R: Puedes hacer un fork del repositorio y enviar un pull request con mejoras.
