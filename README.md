# 30 Ejercicios de Python

Este repositorio contiene 30 ejercicios de programación en Python, organizados por temas.

## Descripción de los ejercicios

### Ejercicios Básicos (1-5)
1. **ejercicio_01.py** - Encontrar el mayor y menor entre tres números
2. **ejercicio_02.py** - Calcular sueldo final con aumentos según rangos
3. **ejercicio_03.py** - Clasificar una persona por edad
4. **ejercicio_04.py** - Calcular promedio de notas y estado del alumno
5. **ejercicio_05.py** - Determinar si es positivo/negativo y par/impar

### Ejercicios de Bucles (6-13)
6. **ejercicio_06.py** - Mostrar múltiplos de 3 entre 1 y 100
7. **ejercicio_07.py** - Mostrar números pares y múltiplos de 5
8. **ejercicio_08.py** - Suma y promedio de 10 números
9. **ejercicio_09.py** - Contar positivos y negativos
10. **ejercicio_10.py** - Encontrar mayor y menor en 10 números
11. **ejercicio_11.py** - Sumar números hasta ingresar 0
12. **ejercicio_12.py** - Calcular promedio hasta ingresar 0
13. **ejercicio_13.py** - Contar positivos hasta ingresar negativo

### Ejercicios de Registro de Datos (14-20)
14. **ejercicio_14.py** - Contar aprobados y desaprobados
15. **ejercicio_15.py** - Promedio y edad mayor
16. **ejercicio_16.py** - Sistema de ventas (5 productos)
17. **ejercicio_17.py** - Sistema de notas para 5 alumnos
18. **ejercicio_18.py** - Sueldo promedio y más alto de empleados
19. **ejercicio_19.py** - Registro de asistencia
20. **ejercicio_20.py** - Contar pares e impares

### Ejercicios con Listas/Arrays (21-28)
21. **ejercicio_21.py** - Cargar y mostrar 10 números
22. **ejercicio_22.py** - Encontrar el mayor en una lista
23. **ejercicio_23.py** - Calcular promedio de una lista
24. **ejercicio_24.py** - Mostrar solo números pares
25. **ejercicio_25.py** - Contar números mayores que el promedio
26. **ejercicio_26.py** - Mayor, menor y promedio
27. **ejercicio_27.py** - Promedio de notas, aprobados y desaprobados
28. **ejercicio_28.py** - Encuesta de edades

### Ejercicios de Lógica (29-30)
29. **ejercicio_29.py** - Mostrar divisores de un número
30. **ejercicio_30.py** - Determinar si un número es primo

## Cómo ejecutar los ejercicios

Para ejecutar cualquier ejercicio:

```bash
python ejercicio_XX.py
```

Reemplaza `XX` con el número del ejercicio que desees ejecutar.

## Requisitos

- Python 3.x instalado en tu sistema
